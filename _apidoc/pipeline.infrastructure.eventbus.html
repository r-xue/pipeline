pipeline.infrastructure.eventbus.eventbus module

The eventbus module defines the interface for publishing events and subscribing to events.

send_message(event: E)[source]

Publish an event.

subscribe(fn: Callable[[E], None], topic: str)[source]

Request callback when a lifecycle event on a topic is published.

The subscribing function should accept a single Event argument.

Parameters:

  • fn – callback function

  • topic – event topic to match

pipeline.infrastructure.eventbus.events module

The events module contains classes representing various stages in the lifecycle of a pipeline reduction.

Events are published on a topic. The implementing class defines at a minimum the event topic. The class may also define additional attributes identifying and/or describing the event. These extra metadata can be interpreted by the event listeners downstream.
