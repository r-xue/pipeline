hifv_vlassmasking(vis=None, vlass_ql_database=None, maskingmode=None, catalog_search_size=None)[source]

hifv_vlassmasking —- Create clean masks for VLASS Single Epoch (SE) images

Create clean masks for VLASS SE images

Output:

results – The results object for the pipeline task is returned.

——— parameter descriptions ———————————————

vis The list of input MeasurementSets. Defaults to the list of MeasurementSets

specified in the h_init or hifv_importdata task.

vlass_ql_database vlass_ql_database - usage in Socorro: /home/vlass/packages/VLASS1Q.fits maskingmode maskingmode options are vlass-se-tier-1 or vlass-se-tier-2 catalog_search_size catalog_search_size in units of degrees
