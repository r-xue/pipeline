class Context(prec=None, rounding=None, Emin=None, Emax=None, capitals=None, clamp=None, flags=None, traps=None)ÔÉÅ

Bases: object

The context affects almost all operations and controls rounding, Over/Underflow, raising of exceptions and much more. A new context can be constructed as follows:

>>> c = Context(prec=28, Emin=-425000000, Emax=425000000,
...             rounding=ROUND_HALF_EVEN, capitals=1, clamp=1,
...             traps=[InvalidOperation, DivisionByZero, Overflow],
...             flags=[])
>>>

power(a, b, modulo=None)ÔÉÅ

Compute a**b. If ‚Äòa‚Äô is negative, then ‚Äòb‚Äô must be integral. The result will be inexact unless ‚Äòa‚Äô is integral and the result is finite and can be expressed exactly in ‚Äòprecision‚Äô digits. In the Python version the result is always correctly rounded, in the C version the result is almost always correctly rounded.

If modulo is given, compute (a**b) % modulo. The following restrictions hold:

  • all three arguments must be integral

  • ‚Äòb‚Äô must be nonnegative

  • at least one of ‚Äòa‚Äô or ‚Äòb‚Äô must be nonzero

  • modulo must be nonzero and less than 10**prec in absolute value
