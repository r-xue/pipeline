calculate(spectrum: List[float], threshold: float = 7.0, tweak: bool = False, mask: List[bool] = [], edge: List[int] | None = None) → List[int][source]ÔÉÅ

Invoke line finding algorithm and return indices of spectral line ranges.

Line detection is performed based on the excess of the median absolute deviation (MAD) of spectrum. Channels of spectrum larger than threshold*MAD are detected as spectral lines. A tweak to line ranges to cover line wings are optionally available.

Parameters:

  • spectrum ‚Äì list of spectrum data.

  • threshold ‚Äì a factor of threshold of line detection with respect to MAD.

  • tweak ‚Äì if True, spectral line ranges are extended to cover line edges.

  • mask ‚Äì mask of spectrum array. An elements of spectrum is valid when the corresponding element of mask is True. If False, invalid. The length of list should be equal to that of spectrum.

  • edge ‚Äì the number of elements in left and right edges of spectrum to be excluded from line detection.

Returns:

A list of start and end indices of spectral lines. The indices of

lines are in the order of, e.g., [start1, end1, ‚Ä¶, startN, endN].

Return type:

ranges
