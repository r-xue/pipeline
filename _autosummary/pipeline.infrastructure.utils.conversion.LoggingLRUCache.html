pipeline.infrastructure.utils.conversion.LoggingLRUCache

class LoggingLRUCache(name: str, *args, **kwargs)[source]

Bases: LRUCache

‘Least recently used’ cache that logs when cache entries are evicted.

Underestimating the required cache size leads to poor performance, as seen in PIPE-327, where a lack of cached entries for the 33 EBs leads to millions of ‘unnecessary’ MS tool open calls, each open() taking several tens of milliseconds. Hence, we want to be notified when the cache size limit is hit.
