FlagDeterBaseInputs manages the inputs for the FlagDeterBase task.

contextÔÉÅ

the (Context) holding all pipeline state

visÔÉÅ

a string or list of strings containing the MS name(s) on which to operate

output_dirÔÉÅ

the directory to which pipeline data should be sent

flagbackupÔÉÅ

a boolean indicating whether whether existing flags should be backed up before new flagging begins.

autocorrÔÉÅ

a boolean indicating whether autocorrelations are to be flagged.

shadowÔÉÅ

a boolean indicating whether shadowed antennas are to be flagged.

toleranceÔÉÅ

a float sets the tolerated projected antenna shadowing in meter. Positive value allows overlap, negative value forces separation.

scanÔÉÅ

a boolean indicating whether scan flagging is to be performed.

scannumberÔÉÅ

A comma-delimited string stating the scans to flag. Standard data selection syntax is valid.

intentsÔÉÅ

A comma-delimited string stating the intents to flag. Wildcards (* character) are allowed.

edgespwÔÉÅ

A boolean stating whether edge channels are flagged.

fracspwÔÉÅ

A float contains the fraction (between 0.0 and 1.0) of channels to removed from the edge.

onlineÔÉÅ

A boolean indicating whether online flags are to be applied.

partialpolÔÉÅ

A boolean stating whether to flag polarization products where part of the polarizations are flagged.

fileonlineÔÉÅ

The filename of the ASCII file containing online flagging commands.

templateÔÉÅ

A boolean indicating whether flagging templates are to be applied.

filetemplateÔÉÅ

The filename of the ASCII file that has the flagging template (for RFI, birdies, telluric lines, etc.).

lowtransÔÉÅ

A boolean stating whether to flag low transmission spectral windows.

mintransnonrepspwsÔÉÅ

A float contains the minimum transmission (between 0.0 and 1.0) required to flag non-representative spectral windows. Defaults to 0.1.

mintransrepspwÔÉÅ

A float contains the minimum transmission (between 0.0 and 1.0) required to flag the representative spectral window. Defaults to 0.05.
