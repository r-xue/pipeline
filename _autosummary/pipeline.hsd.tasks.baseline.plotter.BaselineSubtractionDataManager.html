Create array data for sparse map.

Computes the following masked array data for sparse map: 1. The integrated spectrum averaged over entire spectral data

regardless of their spatial position, which is displayed in the top panel of the figure.

  1. The ‚Äòrepresentative‚Äô spectrum for each panel of the sparse map. rowlist holds the Spatial grouping information of the panels.

  2. Averaged spectra for each panel of the sparse map rowlist holds the Spatial grouping information of the panels. Returned only when produce_averaged_data is specified as True.

Parameters:

  • infile ‚Äì Name of the MS

  • dtrows ‚Äì List of datatable rows

  • num_ra ‚Äì Number of panels along horizontal axis

  • num_dec ‚Äì Number of panels along vertical axis

  • num_chan ‚Äì Number of spectral channels

  • num_pol ‚Äì Number of polarizaionts

  • rowlist ‚Äì List of datatable row ids per sparse map panel with metadata

  • rowmap ‚Äì Row mapping between original (calibrated) MS and the MS specified by infile. It will be EchoDictionary if None is given.

  • integrated_data_storage ‚Äì Storage for integrated spectrum. If None is given, new array is created.

  • map_data_storage ‚Äì Storage for sparse map. If None is given, new array is created.

  • map_mask_storage ‚Äì Storage for sparse map mask. If None is given, new array is created.

  • produce_averaged_data ‚Äì Whether to produce averaged data for each panel. Defaut is False.

Returns:

Integrated spectrum averaged over entire spectral data Representative spectrum for each panel of the sparse map Averaged spectra for each panel of the sparse map (only if produce_averaged_data is True)
