plot(map_data: ndarray | MaskedArray, averaged_data: ndarray | MaskedArray, frequency: ndarray, fit_result: ndarray | None = None, figfile: str | None = None) → bool[source]

Generate sparse profile map.

Generates sparse profile map. If fit_result is given, it is indicated as red lines in sparse profile map. If figfile is provided, the plot is exported to the file.

Parameters:

  • map_data – Data for sparse profile map.

  • averaged_data – Data for integrated spectrum.

  • frequency – Frequency label.

  • fit_result – Data for fit result. Defaults to None.

  • figfile – Name of the plot file. Defaults to None.

Returns:

Whether or not if plot is successful.

Return type:

bool

Set atmospheric transmission data.

If trasnmission and frequency are given properly, atmospheric transmission is overlaid to integrated spectrum as magenta line. Number of elements for transmission and frequency must be the same.

Parameters:

  • transmission – Atmospheric transmission.

  • frequency – Frequency label.

setup_labels_absolute(ralist: List[float], declist: List[float]) → None[source]

Set up position labels.

Set up position labels for both horizontal and vertical axes according to the list of positions along horizontal (ralist) and vertical (declist) axes.

Parameters:

  • ralist – List of horizontal position.

  • declist – List of vertical positions.

setup_labels_relative(refpix_list: Tuple[float, float], refval_list: Tuple[float, float], increment_list: Tuple[float, float]) → None[source]

Set up position labels.

Set up position labels for both horizontal and vertical axes according to reference pixels (refpix_list), reference values (refval_list), and increments (increment_list), which should be given as two-tuples or lists with at least two elements.

Parameters:

  • refpix_list – reference pixels.

  • refval_list – reference values.

  • increment_list – increments.

setup_lines(lines_averaged: List[float], lines_map: List[float] | None = None) → None[source]

Set detected lines.

Provided lines are displayed as shaded area. Lines given to lines_averaged are displayed in the Axes for integrated spectrum. Lines given to lines_map are interpreted as lines for each panel of sparse profile map.

Parameters:

  • lines_averaged – Lines for integrated spectrum.

  • lines_map – Lines for sparse profile map. Defaults to None.

setup_reference_level(level: float | None = 0.0) → None[source]

Set reference level of the sparse profile map.

If float value is given, red horizontal line at the value is displayed to each panel. If None is given, no line is diaplayed.

Parameters:

level – Reference level. Defaults to 0.0.
