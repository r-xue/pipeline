pipeline.domain.state.StateNAOJ

class StateNAOJ(state_id: int, obs_mode: str)[source]

Bases: State

StateNAOJ is a logical representation of (part of) rows in the STATE table for Nobeyama or ASTE Observatory measurement sets, relating STATE_ID (in the MAIN table) to the observing mode(s) and corresponding pipeline intent(s).

Variables:

  • id – Numerical identifier of this State.

  • obs_mode – Unique obs_mode values associated with this State.
