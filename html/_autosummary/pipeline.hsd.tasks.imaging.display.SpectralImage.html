direction_axis(idx: int, unit: str = 'deg') → Tuple[float, float, float][source]

Return conversion information for direction axes.

Three-tuple required for conversion between pixel and world direction axes is returned. The tuple consists of reference pixel, refrence value, and increment for direction axis. Direction index must be given to specify either longitude (0) or latitude (1) axis.

Parameters:

  • idx – Index for direction axes.

  • unit – Direction unit. Defaults to ‘deg’.
